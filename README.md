## Project Overview

Smart Task Manager is a Full-Stack web application developed using **Python Flask**, **SQLite**, **Flask-Login**, and **SQLAlchemy**. It allows users to securely register, log in, and manage their personal tasks efficiently.

The application provides complete CRUD (Create, Read, Update, Delete) functionality with user authentication and a responsive Bootstrap interface.

## Installation

Clone the repository

```bash
git clone https://github.com/Suraj-eng-ux/Smart-Task-Manager.git
```

Move into the project folder

```bash
cd Smart-Task-Manager
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the virtual environment

### Windows

```bash
venv\Scripts\activate
```

### Linux/macOS

```bash
source venv/bin/activate
```

Install the required packages

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser and visit

```
http://127.0.0.1:5000
```
